# Changelog

## [0.1.0] — 2026-06-11

- Initial release
- Core API: show, close, closeAll, shorthand methods
- 4 notification types: info, success, attention, warning
- 6 positions: right-top, left-top, center-top, right-bottom, left-bottom, center-bottom
- Auto-close with pauseOnHover
- Dark theme support (prefers-color-scheme)
- Slide animations with prefers-reduced-motion support
- Full accessibility: role="alert", aria-live, Escape key, focus management
- Zero dependencies
- ESM + IIFE bundle formats
- Bundle size: ~3.6 kB gzip
